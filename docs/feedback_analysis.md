# プロジェクト分析とフィードバック - Summit Gate

## 1. ハッカソン適正評価
**結論: 極めて高い**
- **社会課題の具体性**: 富士山のオーバーツーリズム、山小屋の過酷な労働環境、現金管理という「現場の痛み（Pain）」が非常に明確です。
- **技術の組み合わせ**: IoT（RFID）× ブロックチェーン（JPYC）× 物理ゲートという構成は、単なるアプリ開発に留まらない「社会実装」の説得力があります。
- **スポンサー賞への合致**: 
  - **Avalanche賞**: 高速・低コストな決済、L1構築の可能性（山岳専用チェーン等）をアピールしやすい。
  - **JPYC Innovation**: ステーブルコインを単なる「決済手段」ではなく「分配・DXの基盤」として捉えている点が評価ポイントです。

## 2. JPYCである必然性（Why JPYC?）
「なぜPayPayやクレジットカードではダメなのか？」という質問に対する回答案です。

- **A. 手数料と分配の柔軟性**: 
  - クレジットカード等の3〜5%の手数料は、利益率の低い山小屋運営において大きな負担。JPYC（オンチェーン）ならほぼゼロ。
  - 1,800円を数多くの山小屋へ「利用実績に応じて」銀行振込するのは事務コストが膨大。スマートコントラクトなら自動・即時・安価。
- **B. B2Bエコシステムの完結**:
  - PayPayは消費者から店舗への支払いに特化。山小屋が受け取った売上をそのまま「ブルドーザー代」などの企業間決済（B2B）に回すには、法人間の送金に強いステーブルコインが有利。
- **C. オフライン・プログラム可能性**:
  - 将来的に「システムが自律的に燃料を発注し、JPYCで支払う」といった拡張性は、Web3ネイティブなJPYCならでは。

## 3. 実現性の現実解（委託業者連携戦略）
実績のない個人が行政から直接受託するハードルを、**「既存業者へのシステム提供（B2B2C）」**によって解決します。

- **委託業者のメリット（Why they join?）**:
  - **追加収益**: **「環境保全金（2,000円）」**の徴収を代行することで、手数料収入（例：100円/人）が純増する。
  - **業務効率化**: **日次・週次の一括送金モデル**を採用。都度の送金作業を排除し、業者は既存の会計フローの延長で「溜まった現金を一括でJPYCに換えて送る」だけで済む。
  - **先進性のPR**: 自治体に対し「Web3/IoTを活用した最新の入山管理システムへのアップデート」という実績を報告でき、次期契約の継続に有利に働く。
- **送金メカニズム**:
  - 委託業者が徴収した現金を定期的に一括でJPYCに変換し、スマートコントラクトにデポジットする形式をとる。これにより、業者は「徴収」という得意分野に集中し、Summit Gateが「透明な分配」という難易度の高い部分を自動化する役割分担が可能。

## 4. 技術的面・運用の面
- **通信環境**: 富士山頂付近の電波は不安定。ゲートは「オフラインでカウントし、バッチでオンチェーンへ同期」するアーキテクチャが必須。
- **ハードウェアの堅牢性**: 低温、強風、火山灰。市販のRFIDリーダーでは故障のリスクがあるため、産業用または保護ケースの検討が必要。
- **リストバンド紛失**: 紛失時の再発行フロー、または下山時の罰則規定の明確化。
- **人員配置**: 「3層防御」には結局、入口と出口に人を置く必要がある。これをどう自動化、または既存のスタッフ（組合員）の業務に組み込むかが鍵。

## 5. 行政・法規制の壁
- **徴収の強制力**: 「協力金」ではなく**「環境保全金」**の名目で徴収することで、既存の入山料徴収制度（条例等）と整合性を取り、実質的な強制力を持たせることが可能です。
- **JPYCの取り扱い**: 自治体や組合が直接JPYCを保有・分配することへの会計上のハードル。まずは「民間団体（事務局）が代行する」形での提案が現実的です。

## 6. 全ステークホルダー合意形成上の懸念点と対策

富士吉田モデルを富士宮ルートに導入する際、全関係者（行政・組合・業者）に認知・納得してもらうための主要な課題と対策です。

### 6.1 下山時の「強制」に対する心理的・法的摩擦
- **懸念**: 疲れ切った下山客を呼び止めることへのスタッフの心理的負担と、登山客の反発。
- **対策**: 「強制徴収」ではなく、富士吉田同様に**「環境保全への協力をお願いするゲート」**としてブランディングする。ただし、RFIDゲートによる「未装着者の可視化」を行うことで、心理的な協力圧力を高め、結果として徴収率を最大化する。

### 6.2 運用コストの分担
- **懸念**: 下山口への人員配置によるコスト増。
- **対策**: 既存の「五合目監視員」や「環境指導員」の業務の一部として統合するよう富士宮市・組合に提案。また、委託業者に対しては、下山時の「取りこぼし回収」による増収分で人件費を相殺できる試算を提示する。

### 6.3 山小屋組合の「不公平感」の解消
- **懸念**: 通過数（IoTカウント）による分配に対し、立地条件（八合目と五合目など）による有利不利への不満。
- **対策**: 単純な通過数だけでなく、滞在時間や施設の規模に応じた「重み付け」の検討、または「基本給（一律）＋歩合（通過数）」というハイブリッド分配モデルを提案し、組合内の合意形成を図る。

### 6.4 行政（富士宮市）への「実証実験」としての意義
- **懸念**: 失敗した際のリスク。
- **対策**: 「2026年夏の期間限定」「富士宮ルート限定」というスモールスタートを強調。また、富士吉田という先行事例があることを強力な根拠とし、**「富士吉田にできて、富士宮にできない理由はない」**という行政特有の競争心・横並び意識をポジティブに活用する。

## 7. 想定される質問と回答案（Q&A）

### Q1. 「JPYCでなくても、既存の電子マネー（Suica等）で良いのでは？」

### Q2. 「高齢者の登山客がスマートフォンの操作やウォレットの扱いに困りませんか？」
**A2.** 本プロジェクトの最大の特徴は「ユーザーにWeb3を意識させない」ことです。ユーザーは現金を払ってリストバンドを受け取るだけ（既存のフローと同じ）です。ブロックチェーンの恩恵を受けるのは「山小屋」や「運営側」のバックエンドDXです。

### Q3. 「下山時の検問で未装着者がいた場合、本当に追徴できるのか？（トラブルにならないか）」
**A3.** 吉田ルートでの実績を参考に、物理的なゲートと案内スタッフを配置します。強制的な徴収というよりも「保全への協力」を促すコミュニケーションを重視しつつ、リストバンドがない場合はその場での決済（または後日支払い案内）を促します。

### Q4. 「山小屋が受け取ったJPYCを日本円に戻したいと言われたら？」
**A4.** JPYCは1円単位で日本円建ての商材（燃料・運搬費）の支払いに使えます。また、JPYC社による買取や、二次流通市場での換金ルートを確保することで、キャッシュフローの不安を解消します。実証実験では、事務局が換金をサポートする窓口を設けることも検討します。

### Q5. 「実績のない個人が、どうやって既存の委託業者を動かすのですか？」
**A5.** 本プロジェクトは「業者の既存業務を奪う」のではなく、「業者の収益を増やし、管理コストを下げるためのDXツールを提供する」という立ち位置です。ハッカソンでは、このシステムを導入することで業者が自治体に対して「管理の高度化」をアピールでき、結果として次期契約の獲得率が上がるという、業者側のインセンティブ（三方良し）を強調して提案します。

### Q6. 「数日間〜1週間以上かけてゆっくり登る人（高齢者など）への対応は？」
**A6.** RFIDタグ自体には有効期限を設定せず、システム側で「入山から下山まで」を一つのステータスとして管理します。耐久性の高い素材（強化ビニール等）を採用することで、10日間程度の連続装着でも破損しないようにします。また、このデータは「まだ山の中に誰がいるか」を把握する安全管理データとしても非常に価値が高く、行政や組合にとっても「遭難防止」という強力な導入メリットになります。

### Q7. 「富士宮から入って他ルートへ下山する人はどう管理しますか？」
**A7.** 実証実験フェーズでは、あえて他ルートに回収ボックス等を置かず、**「富士宮に戻ってこなかった数（データギャップ）」を正確に把握すること**に徹します。中途半端な回収は「本当に遭難しているのか、単にタグを出し忘れただけか」の判断を狂わせ、現場を混乱させるリスクがあるためです。この「安否確認の死角」を数値化することで、次シーズン以降の「全山共通システム」へのアップデートの必然性を証明するエビデンスとして活用します。

### Q8. 「スマートコントラクトを導入するメリットは何ですか？」
**A8.** 主に3点あります。
1. **透明性**: どの山小屋にいくら分配されたかがブロックチェーン上に記録され、誰でも検証可能になります（癒着や不明瞭な中抜きの防止）。
2. **自動化と低コスト**: 従来、数百人・数千人の利用実績をExcelで集計し、銀行振込を行うには膨大な事務コストがかかっていました。これをプログラムで自動化することで、振込手数料と人件費をほぼゼロにできます。
3. **B2B経済圏の起点**: 山小屋が受取ったJPYCをそのまま業者への支払いに使えるようになり、「山の上」での資金循環が加速します。

### Q9. 「テクノロジーに頼りすぎて現場が混乱しませんか？」
**A9.** むしろ、現場を「無人化・自動化」するためにテクノロジーを徹底活用します。
- **完全ハンズフリー**: 登山客はゲートを通るだけでよく、スマホ操作やQR提示すら不要です。
- **人的バックアップの廃止**: QRスキャン等の手動運用は人件費を増大させるため、あえて導入しません。按分（実績比率）モデルのため、一時的な停電等による少数のデータ欠落は許容するという割り切った設計にしています。
- **遠隔監視**: 死活監視（ハートビート）とリアルタイム可視化により、現場に行かずとも全ゲートの稼働状況と利用実績を把握できます。

### Q10. 山小屋1,800円の根拠と、業者の納得感はどうでしょうか？
**A10. 現場の実態調査とコスト計算に基づき、以下のロジックで整理しています。**
1. **山小屋1,800円の妥当性**: 富士山では高山病予防のために1日2〜3Lの水分摂取が推奨されており、平均的なトイレ利用回数は1泊2日で5〜8回となります。チップ単価300円で計算すると、6回分（登り4回・下り2回等）の1,800円は、山小屋が提供する環境維持サービスへの対価として極めて適正な水準です。
2. **運営の持続性**: 運営サイドは100円（5.0%）を徴収します。ここからリストバンド原価（50円で固定）を捻出し、残りの50円をハードウェア投資回収、サーバー維持、システム開発費に充当します。原価を固定することで、予算の確実性を担保します。
3. **業者の採算性**: 業者手数料100円（5.0%）は純粋な「現場オペレーション代行」の対価です。リストバンド調達の負担がないため、人件費と決済手数料を差し引いても十分な利益を確保でき、強力な協力インセンティブとなります。
4. **追加収益の確保**: 「銀行振込事務手数料（分配額の5.0%）」を設けることで、JPYC普及を促進しつつ、非利用層（日本円希望者）からの手動事務工数実費を確保し、運営の安定性を高めます。この5.0%は、按分分配された後の金額に対して適用されるため、基本的な分配の公平性は維持されます。
5. **ハードウェアの検知精度**: 屋内設置のRFIDリーダーは、アンテナの指向性を真下に向け、RF出力を調整（絞る）ことで、半径1〜1.5m程度の「真下を通過した時のみ」反応するスポットエリアを作ります。さらに、受信信号強度（RSSI）の低い（＝遠くにある）タグを無視するフィルタリング設定を行うことで、建物の外を歩いている登山客を誤カウントするリスクを排除します。

### Q11. 収集したデータは行政（環境省等）に販売できますか？
**A11.** はい、非常に高いニーズが見込まれます。
1. **環境省のメリット**: 現在、環境省は赤外線カウンター等で登山者数を計測していますが、精度が低く「滞留時間」や「ルートの跨ぎ（トラバース）」までは把握できていません。Summit Gate가提供する「高精度な動態データ」は、国立公園の適正管理（オーバーツーリズム対策）の根拠資料として、**「調査委託」や「データ購入」という形で予算化できる可能性が高い**です。
2. **データの匿名性**: 提供するのは「タグID（ランダムな数値）と通過時刻」のみであり、氏名や住所などの個人情報は含みません。そのため、プライバシー保護を維持したまま、統計資産として販売することが可能です。
3. **新たな収益源**: システム利用料（100円）に加え、このデータ外販を第2の収益源とすることで、開山期間外のシステム開発・保守費用をさらに安定させることができます。

### Q12. RFIDリストバンドの発注は委託業者に任せて大丈夫ですか？
**A12. 結論として、調達・品質管理は「運営（Summit Gate）」が主導します。**
- **技術的リスクの回避**: UHF帯RFIDは周波数設定やアンテナとの相性, 素材の耐久性（9日間以上の装着など）が重要です。非専門の委託業者に発注を任せると、スペック不足による読取エラーが発生するリスクがあるため、運営が一括調達します。
- **委託業者の役割の明確化**: 業者は「現場での配布・装着・説明」という、彼らの強みである「現場運営」に集中してもらいます。### Q13. 50円という低コストで、富士山の過酷な環境に耐えられるリストバンドが本当に作れますか？
**A13. はい、世界的なイベント実績に基づいた高品質なものが調達可能です。**
- **素材の信頼性**: フジロックフェスティバル等の野外フェスや、アイアンマンレース等の過酷なスポーツイベントで採用されている「合成ビニール（PVC）」や「タイベック（高密度ポリエチレン）」を使用します。これらは**防水・防汗・耐引裂性**が非常に高く、意図的に刃物で切らない限り破損しません。
- **長期滞在への対応**: 50円のグレードであれば、接着面を強化することで、ご指摘のあった「9日間以上の滞在」でも剥がれることなく、RFIDの通信性能（読取精度）を維持できます。
- **視認性と偽造防止**: フルカラー印刷が可能（50円に含まれる）なため、ルートごとに色を変えたり、年度ごとのシリアルナンバーや偽造防止用の特殊印刷を施すこともでき、現場での運用強度を大幅に高められます。
